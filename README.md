游쓇릝쓇릝 This is my mini game build by JavaScript 游쓇릝쓇릝

游늶 Rules of this game:



      游리 We have 2 players (game starts with the first player) 
      游리 Each player rolls a die and gets his points (from 2 to 6). This points are not added to main score but to current score.
      游리 Player can roll a die while value is not 1. When it is 1 your current score has value 0 and then you can't add points to your main score.
      游리 If player wants to add current score to main score he must press hold button.
      游리 The player who faster got 100 points on main score wins
